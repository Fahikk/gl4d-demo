# Stars Visualization GL4D

3D graphics visualization of animated stars forming geometric shapes (tori, cubes, spheres) developed with GL4Dummies.

## Description

This project is a real-time 3D rendering demonstration. Thousands of stars are animated in 3D and progressively organize themselves to form various geometric primitives.

## Technology

- **Language** : C
- **Graphics Library** : GL4Dummies (OpenGL wrapper)
- **Shaders** : GLSL
- **GPU Memory Management** : VAO/VBO

## Usage

```bash
# Compile
make

# Run
./main
```
